# Kokkos version of the wire-cell-gen

## prerequisites
 - Need to have access to a WireCell-Toolkit build.
 - Need to have `Kokkos` installed.

## build

```bash
./configure /path/to/kokkos /path/to/install
./wcb install
```

## test

run the kokkos unit test manually
```bash
./build/test_kokkos
```
